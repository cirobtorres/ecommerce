# DJANGO - NEXT.JS BOILERPLATE

## Getting Started

1. Create a folder, name it whatever you wish and, inside that folder, clone this repository like that:

   ```bash
   git clone git@github.com:cirobtorres/django-nextjs-boilerplate.git .
   ```

2. Get inside frontend folder and type:

   ```bash
   npm i
   ```

3. Make sure to rename **.env.example** to **.env** at \backend\dotenv. Do not forget to set up your keys and stablish your database configurations

4. Also rename **.env.local.example** to **.env.local** at \frontend if you wish to make use of OAUTH2 social logins

5. Navigate back to root directory (same as docker-compose) and do as follows:

   ```bash
   docker-compose up
   ```
